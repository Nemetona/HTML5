Html-smörgåsen

Ingredienser:

Html-smörgåsen innehåller grundläggande ingredienser som utgör basen för smakerna

  1. Bröd = Html
  2. smör = Head
  3. sallad = Body

Mellan dessa skivor läggs övrig pålägg som hamnar i valfri ordning = CSS

  • Tomater = bakgrund
  • köttbullar = innehåll
  • rödbetssallad = typsnitt
  • persilja = storlek
